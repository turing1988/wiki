  2. 几个命令的使用
    • bashbug：bash的debug模式
    • dd：转换和复制文件
      • 比如制作启动盘 dd if=iso file path of=/dev/U盘块设备
    • df：显示挂载点使用率
      • -k --block-size=m：使用K M做单位显示各挂载点利用率
    • dmesg：打印或者控制内核缓存信息
      • -c：打印后清除缓存
    • du：打印文件空间利用
    • pidof：打印执行程序PID
    • ping6：测试IPV6网站是否连通
    • ls：罗列目录下文件
      • -l：长信息显示
      • -a：打印所有文件 包含隐藏文件
      • -b：按照c语言风格打印文件既 .c .h格式
      • -c：按照修改时间排序输出
      • -C：按照列输出
      • -d：显示文件夹
      • -h：友好的显示文件大小
      • -i：打印文件的inode节点
      • -k：按照k单位显示文件大小
      • -Q：输出时加上双引号
      • -r：反排序输出
      • -S：按照文件大小排序
      • -s：输出文件大小
      • -t：按照文件新建事件排序
      • -x：按字母排序后按照行输出
      • -X：按照字母表排序
      • -1：一行显示一个文件或目录
    • lsblk：打印块设备
